# Don Magnifico – Laravel Refactor

Questo progetto è un refactor in Laravel di un sito originariamente realizzato in PHP senza framework, attivo per un’attività reale.

## Obiettivo
Il refactor ha l’obiettivo di migliorare la struttura del progetto, la manutenibilità del codice e la gestione dei contenuti tramite database, adottando l’architettura MVC di Laravel.

## Tecnologie utilizzate
- Laravel
- PHP
- MySQL
- Blade

## Contesto
Il progetto nasce dall’esigenza di ristrutturare un codice legacy mantenendo le funzionalità esistenti, rendendo il sistema più ordinato, estendibile e sostenibile nel tempo.

Il repository originale del sito (versione legacy) è mantenuto separato e rappresenta il punto di partenza del refactor.
